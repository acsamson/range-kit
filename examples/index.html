Welcome to Range Kit

This playground demonstrates the capabilities of @life2code/range-kit across different frameworks.

Available Demos
  • Vue Demo: Demonstrates the Vue hooks implementation
  • React Demo: Demonstrates the React hooks implementation
Core Features
  • Selection Restoration: Save and restore user text selections robustly
  • Search Highlighting: High-performance search and highlighting
  • Highlight Navigation: Navigate through highlighted items (like Chrome search)
  • Framework Agnostic Core: Pure TS core logic shared across frameworks
