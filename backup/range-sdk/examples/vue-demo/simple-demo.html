这是一段用于演示文本选区功能的示例文本。Range SDK 是一个强大的文本选择和恢复工具，支持多种选区类型和恢复算法。

你可以选择任意文本片段，然后保存选区。SDK 会自动保存选区信息，并支持在页面刷新或重新加载后恢复选区。

主要特性包括：

  • 精确的文本选区检测和保存
  • 多层恢复算法确保选区准确恢复
  • 支持自定义选区类型和样式
  • 优秀的性能和兼容性

Range SDK 适用于各种需要文本标注和选区恢复的场景，如文档批注、内容审核、学习笔记等。
